# User Action Recorder

A lightweight tool to record and replay mouse and keyboard actions on Windows.

## Features

- Record mouse movements, clicks, scrolls, and key presses
- Accurate replay with timing synchronization
- Loop with fixed iterations or infinite replay
- Custom stop hotkey (default: P)
- Always on top option

## Usage

1. Click **Record** to start capturing actions
2. Click **Stop** when finished
3. Set iterations or enable **Infinite Replay**
4. Click **Replay** to play back
5. Press hotkey to stop early

## Run

```bash
dotnet run
```

## Publish

```bash
dotnet publish -c Release
```
